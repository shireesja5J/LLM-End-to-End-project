# LLM-End-to-End-project
I recently built an LLM End-to-End Blog Generator using Python, Streamlit, and Generative AI.

This application allows users to generate high-quality blog content by simply entering a blog title, keywords, word count, and image preferences. The AI then creates a complete blog along with relevant images in seconds.

🔹 Key Features

✅ AI-Powered Blog Generation
✅ Custom Blog Titles & Keywords
✅ Adjustable Word Count
✅ AI-Generated Images
✅ Interactive Streamlit Interface
✅ Fast & User-Friendly Experience

🛠️ Tech Stack

• Python
• Streamlit
• OpenAI / Gemini API
• Prompt Engineering
• Large Language Models (LLMs)

#Installation & Setup

# Clone Repository
git clone https://github.com/yourusername/LLM-End-to-End-Blog-Generator.git

# Navigate to Project Folder
cd LLM-End-to-End-Blog-Generator

# Create Virtual Environment
python -m venv venv

# Activate Environment
venv\Scripts\activate

# Install Dependencies
pip install -r requirements.txt

# Run Application
streamlit run app.py
